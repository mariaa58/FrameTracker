# FrameTracker

Aplicativo em Python para analisar vídeos frame a frame.

## O que faz

- abre vídeos
- navega quadro a quadro
- permite marcar início, fim e comentários
- salva anotações em `.xlsx`
- salva e reabre projetos em `.json`
- exporta frames como imagem

## Bibliotecas usadas

- Python
- PyQt5
- OpenCV
- openpyxl
- qdarktheme
